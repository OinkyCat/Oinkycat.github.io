Welcome to Quadratic Equations!

Quadratic equations can be tricky and aren't solvable by the rules of equations you have already covered. All quadratic equations form a U or N shape on a graph and are in this format:

ax² + bx + c = 0

There are three ways to solve these equations. The first is by factorisation.

  1. Find a pair of values whose sum is equal to b and whose product is equal to the product of c and a.
  2. If a = 1, rearrange your equation so into a double brackets which each bracket contains x + one of the pair of numbers.
  3. You can then solve the equation to get two values for x.
  • Example:
  • x² + 5x + 6 = 0
  • 2 and 3 have a sum of 6 and a product of 5.
  • (x + 2)(x + 3) = 0
  • If x + 2 = 0, x = -2
  • Otherwise, if x + 3 = 0, then x = -3
